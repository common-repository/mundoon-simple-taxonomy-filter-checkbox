Simple and fast implementation!!

* checkboxes


== Installation ==
= Uploading in WordPress Dashboard =

1. Navigate to the 'Add New' in the plugins dashboard
2. Navigate to the 'Upload' area
3. Select 'mundoon-taxonomy-checkbox-filter.zip' from your computer
4. Click 'Install Now'
5. Activate the plugin in the Plugin dashboard


= Using FTP =
1. Download 'mundoon-taxonomy-checkbox-filter.zip'.
2. Extract the 'mundoon-taxonomy-checkbox-filter.zip' directory to your computer.
3. Upload the 'mundoon-taxonomy-checkbox-filter' directory to the '/wp-content/plugins/' directory.
4. Activate the plugin in the Plugin dashboard

= Configurations =
1. Navigate to the configurations wordpress menu.
2. Click "Taxonomy Filter".
3. Select Taxonomies for you post type.
4. Insert in post type archive file do_shortcode('[mundoon-simple-checkbox-filter cpt="{POSTTYPE}"]');
